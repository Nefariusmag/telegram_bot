--Setup:

Установить: python3 и pip3

Необходимые библиотеки:
pip3 install pytelegrambotapi
pip3 install jenkinsapi

Добавить в крон:
*/15 * * * * cd ~/telegram_bot && ./telegram_jenkins.py
